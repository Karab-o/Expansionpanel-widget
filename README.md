#  Expansion Panel Widget

Flutter app showing an interactive **FAQ screen** to show how the expansion panel widget works with the expansionpanel list as the key since it controls most of the panels.

## Getting Started

## Requirements

- [Flutter](https://docs.flutter.dev/install) (This is in case you don't have flutter installed)
- Dart SDK
- [Android Studio](https://developer.android.com/studio) [to be able to see what your app looks on screen)

## Get to coding..
1. **Clone repository**
    ```
    [git clone https://github.com/karab-o/expansionwidget.git](https://github.com/Karab-o/Expansionpanel-widget.git) ,
                          cd expansionwidget
                          ```
2. **Get dependecies**
   ```
   flutter pub get
   ```
3. **Run App**
   ```
   flutter run
   ```

## Attribute description
**Attribute 1: Expasioncallback**
 It controls what happens when a user clicks a panel, it's like a switch to control opening and closing of the panel.

 **Attribute2: Elevation**
 Adds shadow beneath panel to make it look a bit raised up.

 **Attribute 3: Expandedheaderpadding**
 Adds space around the panels when user clicks to expand to avoid crowding of information around the panel.
